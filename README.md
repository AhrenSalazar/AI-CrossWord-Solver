# AI-CrossWord-Solver

This bot worked from data collected on crossword puzzle clues and answer pairs, sourced from kaggle. 
https://www.kaggle.com/datasets/darinhawley/new-york-times-crossword-clues-answers-19932021

This code was run with Google Colab and so i have imported through google drive as seen.

crossword_model is NOT provided so training is required.
When training for the best runtime use googles T4 CPU, should cut run time down to about 16 minutes.
